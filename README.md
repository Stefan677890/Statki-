Gra w Statki (Battleships) C#

Klasyczna gra w statki (okręty) Celem jest zatopienie wszystkich statków przeciwnika przed tym, jak on zatopi Twoje.

Zasady Gry:

Każdy gracz posiada 10x10 pól.

Flota składa się z:

1x czteromasztowiec

2x trzymasztowce

3x dwumasztowce

4x jednomasztowce

Statki nie mogą się stykać (mogą tylko rogami).

Strzelamy naprzemiennie. Trafienie daje prawo do kolejnego ruchu.

Wygrywa ten, kto pierwszy zatopi wszystkie 10 statków przeciwnika.
